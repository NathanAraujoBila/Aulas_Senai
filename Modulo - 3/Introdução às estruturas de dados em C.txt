📚 Videoaula – Estruturas de Dados

Estruturas de dados servem pra organizar os dados dentro de um programa
Comparação com estrutura de uma casa (organização e sustentação)
Quando os dados estão organizados:
mais fácil de trabalhar
mais eficiente
menos bugs
Tipos de estruturas:
homogêneas (mesmo tipo de dado)
heterogêneas
uma ou mais dimensões
Foco da aula:
estrutura homogênea de 1 dimensão
Significado:
mesmo tipo de dado
armazenado em uma única estrutura
Exemplo:
idade de 30 alunos (todos inteiros → homogêneo)
Importância:
melhora desempenho
organiza memória
facilita algoritmos
